# Rocker

## What is Rocker

Rocker is a lightweight frontend web template built with plain HTML and CSS, ideal for quickly prototyping or launching a simple static website.  
It provides a ready-to-use structure with HTML, CSS styles, fonts, and image assets, so you can focus on customizing content without any setup.

## Why Use Rocker

- Minimal and easy to understand  
- No frameworks or build tools required  
- Simple file structure ideal for beginners  
- Great for landing pages, portfolios, or static sites  
- Easy to clone, fork, and modify  

## Repository Structure

Rocker/
├── index.html ← Main HTML file
├── css/ ← Stylesheets
├── imgs/ ← Images and assets
└── webfonts/ ← Custom font files
